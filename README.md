# Cultiva

Plataforma B2B que conecta pequenos produtores rurais ao varejo por meio de busca inteligente, matchmaking de oferta e demanda e apoio à precificação baseada em dados reais da CONAB.

## Módulos
- [Backend](https://github.com/nahel-argama/PI_III_2026_TB_BAD_THREE_PI/tree/main/backend)
- [Frontend](https://github.com/nahel-argama/PI_III_2026_TB_BAD_THREE_PI/tree/main/frontend)
- Serviços Externos:
  - [Gateway de Pagamento Mockado](https://github.com/nahel-argama/PI_III_2026_TB_BAD_THREE_PI/tree/main/external/payment-gateway)
  - [Scrapper de preços da CONAB](https://github.com/nahel-argama/PI_III_2026_TB_BAD_THREE_PI/tree/main/external/price-scrapper)

## Tecnologias

- PostgreSQL
- Python
- Vue.js
